Sassdoc

SassDoc Package

[WIP] Do not install.

SassDoc compliant comment blocks for SASS/SCSS.

Installing

apm install sassdoc

  • Alternatively, navigate to Settings > Package Manager and search for sassdoc.

Commands

sassdoc:generate

Contributing

The main development branch is develop and the stable ‘production’ branch is master. Please remember to base your branch from master and issue the pull request to the develop branch.
