Img Paste

img-paste

A package to save image in clipboard as a file. and insert markdown img tag for it

Install

  • use apm in terminal

apm install img-paste * or you can also find in Settings/Install by search

How to Use (Linux, Windows)

  1. cut or copy or capture some image
  2. Hit Alt+Ctrl+Shift+V in your editor

How to Use (MAC OS X) 1. cut or copy or capture some image 2. Hit Cmd+Ctrl+Shift+V in your editor
