Linter Plus Eslint

Linter+ ESLint

ESLint interface for linter-plus

How-to / Installation

  • $ apm install linter-plus-eslint

The package will use the locally installed eslint in your project, if it doesn’t find it it will use the eslint installed with the package.

If you use plugins in your .eslintrc you need to have eslint installed locally. ($ npm i --save-dev eslint if it’s not already done).

(Note: In order to have linter-plus-eslint working you will need linter-plus installed)
