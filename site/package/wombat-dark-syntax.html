Wombat Dark Syntax Theme

An Atom syntax theme. ( Python, NodeJS, Markdown, PHP, HTML, CSS )

  • Atom port of the Vim Wombat color scheme.
  • Converted from the One Dark Syntax

Install

This theme is installed by default with Atom and can be activated by going to the Settings > Themes section and selecting it from the Syntax Themes drop-down menu.
