Toolbar Fusion by radjiv fender

Install with: 
    apm install toolbar-fusion

Atom with cool package in a toolbar

Simple tool bar using tool-bar package but with several buttons to aid in software development.

Buttons

  • New File
  • Find and Replace
  • Save all
  • Open recent
  • Git Plus
  • Open on Github
  • Git time machine
  • Terminal
  • Ask StackOverflow
  • Build project
  • Script - Run, run to line, stop run, etc.
  • Preference

Dependencies

  • ask-stack
  • git-plus
  • git-time-machine
  • open-on-github
  • recent-finder
  • terminal-fusion
  • tool-bar

Optional dependencies

  • build
  • script

When in dev mode it adds two more buttons:

  • Reload Window
  • Toggle Developer Tools
