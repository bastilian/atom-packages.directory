Slack UI Alter

A tweaked version of the original slack-ui theme by tony-jones (https://github.com/tony-jones/slack-ui). While tony-jones’ original theme was great, I had a few minor tweaks that I wanted, which is why I forked this out.

A light UI theme for Atom that is inspired by Slack’s aubergine theme. https://atom.io/themes/slack-ui ###### Made by @iamtonybagels

Installation

Fire up a console and type:

$ apm install slack-ui

Or, inside Atom’s settings select Install and then search for this package.

Configuration

Use with Atom’s ‘Atom Light’ Syntax theme.
