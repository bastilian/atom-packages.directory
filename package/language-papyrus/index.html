Papyrus

Install with: 
    apm install language-papyrus

language-papyrus package for Atom

Adds support for the Papyrus language and its assembly to Atom.
For Papyrus compilation, use the build and build-papyrus packages.

Papyrus is a scripting language for the Creation Engine, the game engine that The Elder Scrolls V: Skyrim and Fallout 4 run on. Mods for Skyrim and Fallout 4 use this language to make things happen in the game. Want to learn more about it? Check out the Papyrus reference for Skyrim or Fallout 4.

Grammars and snippets for Skyrim are based on SublimePapyrus, converted for Atom and modified.
