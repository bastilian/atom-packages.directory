Install with: 
    apm install lucid-syntax

lucid-syntax

An easily customisable syntax theme for Atom that employs a minimum set of colours to optimise semantic clarity.

Motivation

I created this theme after being discontent with using many other popular syntax themes for Atom that:

  1. Employ colours too liberally, which I feel is similar in clarity to reading a file with no syntax colouring at all.
  2. Provide an impressive number of syntax colouring hooks, which can be extremely time consuming to get right if simply one wants to add one or two tweaks.

Usage

Simply install this package via the Install tab in the Settings View for Atom and select it to be the syntax theme using the dropdown in the section opened by the Theme tab.

If you wish to customise this theme to your own liking, please see the styles/custom.less file for guidance.

Please note that this theme is specifically reviewed for:

  • C++
  • CSS (SCSS, LESS)
  • HTML
  • JavaScript
  • Markdown (GFM)
  • Rust

It has also been tested against the default Atom UI themes (Atom Dark, Atom Light, One Dark, One Light) to ensure seamlessness.

Notes

Automatically selected colours based off the primary colour follows the split complementary colour scheme plus the complementary colour.
