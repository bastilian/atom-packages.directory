Tinacious Design Syntax by Tina Holly

Install with: 
    apm install tinacious-design-syntax

Tinacious Design Syntax is a dark syntax theme that uses a bright palette of colours including pink, blue, turquoise, green, purple, and orange.

Colours curated by Web and Mobile designer and developer Tina from Tinacious Design.

Complements the One Dark UI theme

When used with the One Dark UI theme, this syntax theme offers a dark gray UI with a subtle purple tint.

The font used in the Atom editor is Inconsolata, and the tab bar, status bar, and tree view font is Source Code Pro.
