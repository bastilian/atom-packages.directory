Android Debugger

Install with: 
    apm install android-debugger

atom-android-debugger package

This is a fork from https://github.com/xndcn/atom-debugger

Before debugging, please add your source root directory into the tree-view and start gdbserver from your device:

adb forward tcp:5039 tcp:5039 adb shell /system/bin/gdbserver --multi localhost:5039
