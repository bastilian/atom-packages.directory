Custom Minimap Titles

Atom package to convert the selected text to ASCII art using FIGlet and then comment it.

The chosen figlet font (Banner3) is perfect for minimap visualization. If you want to change the font, add the following into your Atom config: "custom-minimap-titles": font: "Small" All the available fonts can be found at http://patorjk.com/software/taag

Usage

To convert text, select the text you wish to convert, and do any of the following: - Press ctrl-shift-del or - Select Minimap Titles: Convert in the command palette or - Select Packages -> Minimap Titles -> Convert from the main menu or - Right click on the selected text, and select Minimap Titles: Convert from the context menu

To create comment borders around text to help distinguish sections in the minimap do the following: - Select Packages -> Minimap Titles -> Toggle Comment Borders from the main menu - Run the Convert command on your text as detailed above
