Prisma Syntax

Install with: 
    apm install prisma-syntax

To Install

Install using Atom - Go to Atom -> Preferences… - Then select the Install tab - Click the Themes button to the right of the search box - Enter prisma-syntax in the search box

Install Using Git - If you are a git user, you can install the theme and keep up to date by cloning the repo directly into your ~/.atom/packages directory.

Install Using Command Line - You can also install right from the command line by running npm install prisma-syntax

Install Manually - Download using the GitHub .zip download option and unzip them - Move the dracula-theme folder to ~/.atom/packages

Activating theme - Go to Atom -> Preferences…, click in the Themes tab, and select Prisma in the Syntax Theme dropdown.
